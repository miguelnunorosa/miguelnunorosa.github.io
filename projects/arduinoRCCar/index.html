Arduino RC Car

Esta página é dedicada ao projecto que desenvolvi em Arduino, a criação de um RC Car controlado por bluetooth.

Sobre o Projecto

Este projecto foi criado para a disciplina de Física Aplicada à Computação (Instituto Politécnico Beja - Engenharia Informática).

O desafio consistia em criar um RC Car, controlado via bluetooth utilizando para tal um dispositivo móvel (a opção foi o sistema Android, na versão (igual ou superior) 5.0 - Lollipop).
Neste espaço estão disponíveis os esquemas, download da app para controlar o carro bem como todo o código fonte.

Foram utilizados os seguintes materiais para a construção física do RC:

  • Arduino Uno R3
  • Sensor Ultra-sons HC-SR04
  • Sensor LDR (light Dependent Resistor)
  • Sensor Temperatura LM35
  • Módulo Bluetooth HC-06
  • Ponte H
  • Resistências
  • LED's
  • Motores DC com redutor
  • Buzzer
  • 8x Pilhas 1.5v 1200mA (fazendo o total de 12v)

Para a programação foi usado o IDE Visual Studio (com plugin Visual Micro) na programação do sketch arduino e o IDE Android Studio na programação da aplicação Android para controlar o RC.
